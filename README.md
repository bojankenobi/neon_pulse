# âš¡ Neon Pulse: Grid Combat

**Neon Pulse** is an ultra-fast arcade combat game inspired by retro-futuristic aesthetics, 
developed as a **PWA (Progressive Web App)** for a premium experience across mobile and desktop devices. 
This project showcases advanced Canvas rendering techniques, performance optimization, and immersive audio design.

## ğŸš€ Key Features

- **High Performance:** Optimized Renderer maintaining a stable 60 FPS, even in Squad Mode (1vs3).
- **PWA Ready:** Installable directly to the home screen without an app store, featuring a custom manifest and service worker.
- **Haptic Feedback:** Integrated vibration support for steering and collisions on mobile devices.
- **Dynamic Audio:** Utilizes the `Web Audio API` for high-fidelity .wav samples with a real-time Pitch Shift effect during Turbo mode.
- **Responsive 3D Grid:** Adaptive perspective scaling for various screen dimensions, from ultra-wide monitors to flagship mobile devices like the Samsung S20 Ultra.

* **Language:** JavaScript (ES6+ Modules)

* **Storage:** Service Workers & Cache API (Full offline support)
* **Styling:** CSS3 with neon keyframe animations and perspective transforms
